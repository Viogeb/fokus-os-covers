# FOKUS.OS Cover Images

Eigene, unbearbeitet fotografierte Bilder (Copyright bei der Autorin), farblich auf die FOKUS.OS-Markenpalette (Cream/Terracotta) abgestimmt. Dient als öffentlich erreichbare Bildquelle für Notion-Seiten-Cover (Notion "external cover" braucht eine öffentliche URL).

Ersetzt die vorherige Unsplash-Hotlink-Lösung wegen Lizenz-Unklarheit bei eingebetteten Fotos in verkauften digitalen Templates.
